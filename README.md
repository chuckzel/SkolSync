# SkolSync

A configurable C# library for synchronization of external systems and provisioning resources, runnable locally from a .NET file-based app.

> ⚠️ **Work in progress**  
> The codebase is currently under active development and may change frequently.

## Example usage

This project is still early and not yet stabilized.

Usage showcasing some current capabilities (synchronizing users from an on-premises information system database with Active Directory users) is available [in branch zstlum](https://github.com/chuckzel/SkolSync/blob/zstlum/zstlum/zstlum.cs)
